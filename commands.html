How to Use Commands

All commands can be used in your Discord server where the bot is installed. Simply type the command in any channel where the bot has permissions.

Note: Replace <clan_tag> and <player_tag> with actual tags. The # symbol is optional.

Additional Information
Bot Setup

The bot requires proper permissions in your Discord server to function correctly:

  • Read Messages
  • Send Messages
  • Embed Links
  • Use External Emojis
  • Add Reactions
Data Sources

All data is fetched in real-time from the official Clash of Clans API:

  • Clan information and member lists
  • Player statistics and achievements
  • War and CWL data
  • Troop, spell, and hero levels
Rate Limits

Commands may have cooldowns to prevent spam and respect API limits. Please wait a moment between requests.

Privacy

The bot only accesses publicly available data from the Clash of Clans API. No private information is stored or shared.
